# 👥 KeenKeeper — Keep Your Friendships Alive

## 📌 Overview

**KeenKeeper** is a modern friendship management web app that helps users maintain meaningful relationships by tracking interactions like calls, texts, and video chats.

It ensures you never lose touch with your important people by providing reminders, timelines, and analytics.

## 🛠️ Technologies Used

- ⚛️ Next.js (App Router)
- 🎨 Tailwind CSS + DaisyUI
- 🔥 React Hot Toast (Notifications)
- 📊 Recharts (Analytics)
- 🌐 JSON (Local Data Handling)

---

## ✨ Key Features

### 👥 Friend Management

- View all friends in a clean card layout
- Each friend shows:
  - Profile image
  - Name
  - Days since last contact
  - Tags
  - Status (Overdue / Almost Due / On Track)

---

### 📞 Quick Check-In System

- One-click interaction buttons:
  - 📞 Call
  - 💬 Text
  - 🎥 Video

- Automatically logs interactions into timeline
- Shows instant toast notifications

---

### 📜 Timeline Tracking

- Displays all interaction history
- Includes:
  - Date
  - Interaction type (icon)
  - Title (e.g., "Call with John")

- Filter by:
  - Call / Text / Video

---

### 📊 Friendship Analytics

- Pie chart visualization of:
  - Calls
  - Texts
  - Video interactions

- Helps analyze communication habits

---

### 📄 Friend Details Page

- Detailed friend profile with:
  - Bio
  - Email
  - Tags
  - Status

- Stats section:
  - Days since contact
  - Goal
  - Next due date

- Action buttons:
  - Snooze
  - Archive
  - Delete

---

### 📱 Fully Responsive Design

- Works smoothly on:
  - Mobile 📱
  - Tablet 💻
  - Desktop 🖥️

---

### ⚡ Additional Features

- 🔄 Loading animation while fetching data

- 🔔 Toast notifications for interactions
- 🔗 Smooth routing with Next.js App Router
